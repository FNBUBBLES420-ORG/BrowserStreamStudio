# BrowserStream Studio 1.2.10

## Windows Installer

Download and run:

`BrowserStream Studio Setup 1.2.10.exe`

## Included In This Build

- Desktop streaming app for Windows
- Built-in local backend
- Scene and stream controls
- Recording browser
- Account connection flow for supported providers
- Plugin support

## Before You Open The App

- Windows may ask for permission to run the installer
- FFmpeg and FFprobe are bundled with this Windows build

## Install Steps

1. Open `BrowserStream Studio Setup 1.2.10.exe`
2. Follow the installer steps
3. Finish setup
4. Launch BrowserStream Studio from the desktop or Start menu

## Notes

- This release is for Windows
- If SmartScreen appears, review the prompt and continue only if you trust the file source
