# Contributing to BrowserStreamStudio

Thank you for your interest in contributing! We welcome all kinds of contributions, including bug reports, feature requests, documentation improvements, and code changes.

## How to Contribute

1. **Fork the repository** and create your branch from `main`.
2. **Describe your changes** clearly in your pull request.
3. **Follow the coding style** and conventions used in the project.
4. **Test your changes** before submitting a pull request.
5. **Sign your commits** if required by the project.

## Reporting Issues

- Use the GitHub Issues tab to report bugs or request features.
- Please provide as much detail as possible, including steps to reproduce and screenshots if relevant.

## Code of Conduct

Please be respectful and considerate in all interactions. See our CODE_OF_CONDUCT.md for details.

## Getting Help

If you have questions, open an issue or join our discussions tab.
